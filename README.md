
# 🚀 Forge Fitness

Uma plataforma web para gerenciamento de academia, permitindo o cadastro de alunos, treinos e acompanhamento de desempenho.

![Capa do Projeto](./assets/Biblioteca.png)

## ✨ Funcionalidades

- 👤 Cadastro de usuários
- 💪 Gerenciamento de treinos
- 📅 Controle de frequência
- 📊 Dashboard
- 🔐 Sistema de autenticação

## 🛠️ Tecnologias

- Node.js
- Express
- EJS
- CSS
- JavaScript
- MySQL (ou o banco que estiver usando)


## 🚀 Como executar

```bash
git clone https://github.com/seu-usuario/forge-fitness.git

cd forge-fitness

npm install

npm start
```

Depois acesse:

```
http://localhost:3000
```

index.js
